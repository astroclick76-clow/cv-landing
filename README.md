## 🎨 Personalización

### Cambiar información personal:
1. **Nombre y título**: Editar `<h1>` y `<h2>` en la sección Hero
2. **Descripción**: Modificar el texto en `.hero-description`
3. **Experiencia**: Editar los cards en `.experience-card`
4. **Habilidades**: Modificar porcentajes en `.skill-level`
5. **Contacto**: Actualizar información en `.contact-item`

### Cambiar colores:
Modificar las variables CSS en `:root`:
```css
:root {
    --primary: #2c3e50;      /* Color principal */
    --secondary: #3498db;    /* Color secundario */
    --accent: #e74c3c;       /* Color de acento */
    --light: #ecf0f1;        /* Color claro */
    --dark: #1a252f;         /* Color oscuro */
}
```
